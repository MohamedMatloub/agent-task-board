# Board

Local Markdown project board.

## Read Order

1. `status.md`
2. Relevant `features/<feature>/<epic>/TASK-000.md`
3. Parent `EPIC.md` or `FEATURE.md` only when needed

## Layout

```text
features/<feature>/FEATURE.md
features/<feature>/<epic>/EPIC.md
features/<feature>/<epic>/TASK-000.md
done/
changelog.md
decisions.md
```
